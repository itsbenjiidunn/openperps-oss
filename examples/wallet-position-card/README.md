# wallet-position-card

This example shows how to integrate OpenPerps through `@openperps/sdk`.

## What It Demonstrates

- list a wallet's OpenPerps positions
- show PnL
- show a liquidation price estimate if available

## Run

```bash
npm install
npm run dev      # or: npm run build
```

`src/App.tsx` is the whole integration, built on `@openperps/react`.

## Boundaries

This example targets Solana devnet.
